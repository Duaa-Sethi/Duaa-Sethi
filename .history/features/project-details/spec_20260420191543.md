# Feature: Project Details Page

## Goal

Implement index.html which is the main landing page of my portfolio.

## Designs

![target layout](../../assets/IMG_4092.jpeg)

## Inspiration

I took inspiration from kentcdodds.com because I really liked how clean and light it looks.
It has a big bold name at the top, simple navigation and a warm friendly feel which is the
vibe I want for my portfolio too.

## Work

- A navbar at the top with my name on the left and links on the right (About, Projects, Contact)
- A hero section with my name in big bold text and a short tagline like "CS Student and Aspiring Developer"
- An about section with a short bio about me and a photo
- A projects section with cards showing the project title, short description, what tech I used and a GitHub link
- A footer with my GitHub, LinkedIn and email
- Hover behavior: nav links get an underline when you hover, project cards lift up a little with a light shadow

## Deliverables

- [ ] index.html exists and opens in the browser
- [ ] Navbar shows at the top and the links work
- [ ] Hero section shows my name and tagline
- [ ] At least 2 project cards are showing
- [ ] Footer has my social links
- [ ] Hover effects work on the nav links and project cards
